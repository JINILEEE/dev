기본문법

1. 변수와 자료형

        변수 선언
            var : 동일한 변수이름을 여러개 넣어도 오류가 안뜬다! 선언을 2번 이상 할 수 있다. (중복 선언 가능) 
                  최대한 안쓰는게 좋다. (console.log()보다 밑에서 실행되어도 에러가 안뜨는 이상한 상황이 발생한다...)
            let : 우리가 알고있는 일반적인 변수를 만들어준다.
            const : 자바에서 'final 변수'와 같다. 재할당이 불가능하게 만들어줄 때 사용.

        자료형
            string : 문자열
            number : 모든 숫자 처리
            boolean : true / false 들어가는 논리형 변수
            object : 객체 / key-value 형태로 들어감
            function : 함수
            undefined : 초기화가 안된 변수

2. 데이터 형변환


        문자열 + 숫자 => 문자열
        숫자 + 숫자 + 숫자 + 문자열 => 문자열

        [강제 형변환]
        parseInt()
        parseFloat()
        Number()

        String()
