배열

        JS 는 타입이 없으므로, 타입 구분없이 데이터를 담을 수 있다.
        크기 지정 필요 없음

1. 선언 및 초기화

2. 메소드

        
        indexOf() - 특정 문자 위치 찾기 /  배열에서 요소가 위치한 인덱스를 반환한다.
        concat() - 문자 배열을 합칠때 사용하는 함수 api
        join() - 배열의 모든 요소를 연결해 하나의 문자열로 만든다.
        reverse() - 문자배열 순서를 역순으로 뒤집어준다.
        sort() - 문자열 기준으로 오름차순으로 정렬
        push() - 배열의 마지막에 새로운 요소를 추가한 후, 변경된 배열의 길이를 반환
        pop() - 배열의 마지막 요소를 제거한 후, 제거한 요소를 반환
        shift() - 배열의 첫 번째 요소를 제거한 후, 제거한 요소를 반환
        unshift() - 배열의 첫 번째 자리에 새로운 요소를 추가한 후, 변경된 배열의 길이를 반환
        slice() - 배열의 특정 위치에 배열 요소를 추가하거나 삭제하는데 사용합니다. 
                  리턴값은 삭제한 배열 요소입니다. 삭제한 요소가 없더라도 빈 배열을 반환합니다.
        splice() - 특정 인덱스 위치에 요소 추가/삭제
